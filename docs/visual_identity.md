# Визуальная идентичность · CDP Generation

## Концепция

**Премиум-аналитика с морским характером.** Фиолетовая сила бренда Calltouch + глубина и спокойствие морского cyan. Палитра, которая считывается одновременно как «технологичный SaaS» и как «премиум-сегмент».

## Логотип Calltouch в документах

**Размещение:**
- Обложка PDF — верхний левый угол, моноширинный лейбл `CALLTOUCH · CDP` (буква-сэмплирование, не растровое лого)
- Подвал каждой страницы — `CDP Calltouch · {client_name} · {year}` мелким шрифтом справа
- Веб-интерфейс — лого в шапке, кликабельное

**Безопасная зона:** минимум 12 pt отступа со всех сторон от лого.

## UI-компоненты (фаза 1, Streamlit)

Streamlit поддерживает кастомизацию темы через `.streamlit/config.toml`:

```toml
[theme]
primaryColor = "#5B2FBF"
backgroundColor = "#FFFFFF"
secondaryBackgroundColor = "#F2EDFF"
textColor = "#0F172A"
font = "sans serif"
```

Дополнительные элементы (карточки, бейджи) — через кастомный CSS в `st.markdown(..., unsafe_allow_html=True)`.

## PDF-шаблон

Базовая HTML-структура для PDF унаследована от существующих кейсов LW Group и Level Group, расположенных в:
- `../cdp_calltouch_kb/output_lw_group/LW_Group_CDP_Proposal.html`
- `../cdp_calltouch_kb/output_level_group/Level_Group_CDP_Proposal.html`

Эти файлы используются как **эталонные шаблоны**. При генерации новых КП — берём за основу, заменяем содержимое, сохраняем визуальную идентичность.

**Ключевые элементы шаблона:**
- Обложка с градиентом `--gradient-deep` и золотым акцентом для премиум-клиентов либо `--gradient-brand` (фиолет → cyan) для остальных
- Карточки сценариев с фиолетовым номером в круге
- Метрики в виде сетки 3×N с большой цифрой и капсом-подписью
- Финальный блок «Состав решения и стоимость» в фиолетовой обводке

## Что НЕ делать

- ❌ Использовать яркий красный, кислотные цвета, неон — это премиум-сегмент
- ❌ Брать растровое лого Calltouch вместо моноширинного лейбла без согласования с менеджером (право на лого регламентируется отдельно)
- ❌ Перегружать страницу — премиум любит воздух
- ❌ Использовать рукописные шрифты, готику, декоративные шрифты
- ❌ Использовать стоковые иконки emoji в финальных PDF (только в служебных интерфейсах)
