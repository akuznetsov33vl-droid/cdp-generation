# Идея проекта · одна страница

## Что мы делаем

**CDP Generation** — приватный AI-инструмент для одного человека (продающий менеджер Calltouch), который автоматически генерирует индивидуальные коммерческие предложения по CDP под конкретных клиентов.

## Зачем

Сейчас процесс ручной: менеджер изучает сайт клиента, расшифровку звонка, базу знаний, потом несколько часов собирает PDF с уникальными сценариями. На каждого клиента — 3–5 часов.

Автоматизация снижает время до 5–10 минут на одно КП и при этом сохраняет уровень качества «как руками» — потому что вся экспертиза уже формализована в локальной базе знаний `cdp_calltouch_kb/`.

## На вход

- URL сайта клиента (один или несколько доменов)
- Бриф / транскрипт встречи / краткое описание задачи
- Опционально: дополнительные пожелания менеджера

## На выходе

- HTML и PDF с индивидуальным предложением:
  - Понимание клиента и сегментация ЦА
  - 3–10 уникальных сценариев CDP под архитектуру Calltouch
  - Релевантные кейсы из базы знаний
  - Состав решения и стоимость по актуальным ценам
  - План запуска

## Главные принципы

1. **Реализуемость:** только триггеры и действия, реально доступные в CDP Calltouch
2. **Соответствие правилам:** актуальные цены, без упоминания конкурентов по своей инициативе, без запрещённых категорий
3. **Конкретика клиента:** реальные имена с сайта, реальные проекты/услуги, реальные URL-паттерны
4. **Премиум-стиль:** фиолетовый + морской cyan, фирменный логотип, чистая типографика

## Развитие проекта

- **Фаза 1 (сейчас):** локальный инструмент на машине пользователя
- **Фаза 2:** перенос в личное облако, авторизация по Telegram, БД на Supabase, деплой через GitHub
