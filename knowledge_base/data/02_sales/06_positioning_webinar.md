# 6. Позиционирование Смарт-коммуникаций (вебинар для сотрудников)

> Источник: внутренний вебинар Calltouch «Позиционирование Смарт-коммуникаций. Место продукта в мире CDP и автоматизации маркетинга». Этот документ — главный источник для понимания позиционирования продукта против конкурентов и для обоснования продаж клиентам.

## 🎯 Главная идея

> **Смарт-коммуникации (CDP Calltouch) — единственная экосистема для маркетологов**: инструменты для каждого этапа воронки (колтрекинг, сквозная аналитика, аналитика на Big Data, работа с конверсией) в одном продукте.

## Целевая аудитория Колтач

| Параметр | Значение |
|----------|----------|
| **Сегмент бизнеса** | СМБ B2C, не имеющие CDP |
| **Сферы (приоритет)** | С длительным циклом сделки или звонковые тематики |
| **Core-отрасли (лёгкий выход на ЛПР)** | Автодилеры, клиники, застройщики |
| **Отрасли с повторными продажами** | Ecom, авто, фитнес, мед. клиники |

## Преимущества Колтач (6 ключевых)

1. **Сценарии под ключ** — с клиента цели и задачи, с Колтач настройка экспертами
2. **Интеграции бесплатно** — подключим CRM, сайт и сервисы рассылок за наш счёт
3. **Быстрый старт с Big Data** — запускайте коммуникации без ожидания накопления данных
4. **Уникальная дата** — знаем по Big Data партнёров, что клиент интересуется конкурентом, и поможем вернуть его
5. **Сквозная аналитика** — отслеживайте весь путь, от касаний до ROI
6. **Готовые отраслевые сценарии** — выбирайте подходящие механики из 10+ карт коммуникаций

## Роли клиентов (для кого CDP)

- Маркетолог
- Email-маркетолог
- CRM-маркетолог
- Руководитель отдела продаж
- Владелец компании

## Проблематика клиентов

CDP решает 5 ключевых проблем:
1. Посетители слабо конвертируются в лиды
2. Лиды не конвертируются в продажи
3. Сделки долго движутся по воронке
4. Средний чек ниже расходов на привлечение
5. Мало повторных продаж

## 📋 Главные выводы (для презентации клиенту)

### 1. Позиционирование Калтач

> **«Лёгкое решение для старта работы с базой»**

- Mindbox и Altcraft — это «полноценный CRM-маркетинг» для **средних и крупных**
- Калтач — это **первый шаг** в работу с базой для СМБ

### 2. Уникальность

> **«Калтач — единственная экосистема для маркетологов»**

Инструменты для каждого этапа воронки в одном продукте:
- Колтрекинг (отслеживание звонков)
- Сквозная аналитика (от касания до ROI)
- Аналитика на Big Data (партнёры)
- Работа с конверсией (CDP / Смарт-коммуникации)

> Подтверждение: интервью с CRM-агентствами показывает, что **пользователи хотят триггерную платформу + сквозняк в одном окне**.

### 3. Отсутствие функций — не проблема

Например, **продуктовые рекомендации** на сайте сейчас не нужны — у Калтач нет глубокой интеграции с Ecom. Это **вектор развития**, а не недостаток.

### 4. Чем отличаемся от каждой категории

#### Прямые конкуренты
- Отсутствие функций — это вектор развития под нашу ЦА
- Разное позиционирование — Калтач это лёгкое решение для старта
- **Единственная экосистема** среди представленных

#### Вторичные конкуренты
- Задачи те же, но узкие. CRM двигает по воронке, рассылки — поддерживают коммуникацию. Смарт-коммуникации **помогают на любом шаге**
- Поднять CR сайта, вернуть после недозвонов, генерация лидов и движение по воронке
- Open rate и click rate триггеров **в 2 раза выше** массовых рассылок

#### Косвенные конкуренты
- Они либо **придут в ТКБ**, либо **закроются** — экономика лидгена перестаёт сходиться

## 🚀 Вектор развития продукта

1. **Переход в CDP или CDxP** — смена нейминга и позиционирования (с «Смарт-коммуникаций» на «CDP Calltouch»)
2. **Партнёрки с CRM-агентствами** — должны стать основным каналом дистрибуции
3. **Чёткое выделение ЦА** — без этого «ориентация на всех = ориентация ни на кого»
