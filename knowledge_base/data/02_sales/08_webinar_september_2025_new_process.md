## 🎯 Главные изменения с конца 2025

### 1. Новая структура отдела работы с клиентами

**До ноября-декабря 2025:**
```
Менеджер по продажам → Продаёт ВСЁ:
- Идею продукта
- Конкретные сценарии
- Технические детали
- Договорённости по интеграции
- Контентную часть
```

**С ноября-декабря 2025:**
```
Менеджер по продажам → Только продаёт идею
                  ↓
         (после оплаты счёта)
                  ↓
Отдел бизнес-аналитики (ОБА) → Берёт ведение проекта:
- Сегментирует аудиторию
- Настраивает триггеры
- Готовит медиаплан
- Делает срез эффективности
```

### 2. Что это даёт менеджеру по продажам

> «На встрече надо продать **идею** Smart-коммуникаций. Не нужно много деталей. Нужно показать все задачи клиента, что мы можем эти проблемы решать, "занеси бабло — мы всё сделаем", оставь бриф. **На этом миссия менеджера по продажам заканчивается.**»

**Преимущества:**
- ✅ Не нужно глубокое знание технических нюансов
- ✅ Не нужно вникать в контентную часть
- ✅ Простая и понятная роль на встрече
- ✅ После оплаты — ОБА сами доводят до запуска

### 3. Медиаплан — новый этап в воронке (с ноября-декабря)

**Что такое медиаплан:**
- Документ, подготовленный отделом бизнес-аналитики
- Содержит конкретные сценарии для клиента
- Опирается на калькулятор маржинальности
- Показывает прогнозные затраты, выгоды, профит

**Когда готовить:**
- Можно **до получения оплаты** (для дожима клиента)
- Это не правило «только после оплаты»

**Как использовать на возражения:**
```
Клиент: «Какие-то невпечатляющие у вас сценарии, скучно всё»
Менеджер: «OK, не вопрос. Я отправлю запрос в бизнес-аналитику.
          Наши эксперты на основе вашего бизнеса подготовят
          релевантное решение. Необязательно решать на встрече —
          просто заполните бриф.»
```

> «Чем больше людей на следующий этап протолкнём — тем больше будет конверсия в итоговые продажи.»

### 4. Когда медиаплан НЕ нужен

> «Если ты чувствуешь силу подключить конкретное решение, порекомендовать и закомметиться — пожалуйста, никаких вопросов нет. Передаётся в ОБА конкретная инициатива, ОБА докручивают, реализовывают, запускают. **Медиаплан ради медиаплана нам нахрен не нужен.**»

## 🧮 Калькулятор маржинальности — детальная инструкция

### Где найти

```
calltouch.ru → Продукты → Смарт-коммуникации
              ↓
   Прокрутить страницу до середины/конца
              ↓
        Калькулятор маржинальности
```

### Базовая версия (для быстрого расчёта на встрече)

**Шаги:**
1. Нажать «Рассчитать»
2. Указать отрасль (например, медклиника)
3. Указать посещаемость (например, 20 000 визитов)
4. Нажать «Рассчитать»

**Что выводит:**

| Параметр | Что показывает |
|----------|----------------|
| CR в лиды | Среднеотраслевая (например, 3% для медклиники) |
| Валовые лиды | Среднее число (например, 600 в месяц) |
| CPL | Среднеотраслевой (например, 250 ₽) |
| Средний чек | Среднеотраслевой (например, 3 500 ₽) |
| **Абонентская плата** | 8 500 ₽ |
| **Доп. лидов с CDP** | Ассоциированные (например, 216) |
| **Цена доп. лида** | Например, 39 ₽ |
| **Доп. выручка** | Конкретная цифра |

### Pro-версия (на основе данных клиента)

**Шаги:**
1. Указать отрасль
2. Указать **свою** посещаемость
3. Указать **свои** лиды в месяц
4. Указать **свой** CPL
5. Указать **свой** средний чек

**Результат:** более точный просчёт на основе реальных данных клиента.

### Важные оговорки

> ⚠️ «Расчёты примерные, основанные на среднеотраслевых показателях. **Это не публичная оферта.** Мы не комитимся на эти результаты.»

> «Это для демонстрации того, что мы знаем, на что и как мы влияем с помощью внедрения Smart-коммуникаций. Показываем это на цифрах.»

### Учёт стоимости Скоринга в калькуляторе

**В калькуляторе — НЕ учитывается.** Только базовая абонка.

> «Прибавляется к 8 500. Средняя цена за Скоринг — десятка по минималке. Совместное решение, совместная абонка. **Маржа, которая показывается в калькуляторе, обычно больше — соответственно, можно не опасаться накидывать сразу наход.**»

## 🎓 Главные выводы из вебинара

### Для менеджеров по продажам

1. **Продавайте идею, не детали** — сложные вопросы передавайте в ОБА
2. **Используйте калькулятор маржинальности** — это самый сильный инструмент дожима
3. **Карты коммуникаций — для экспертности**, не для выбора решений на встрече
4. **Медиаплан станет основой работы** с возражениями типа «непонятные сценарии»
5. **Не бойтесь возражений** — большинство снимаются через «делаем под ключ»

3. **Технические детали — не ваша зона** в новой структуре

3. **Презентация теперь полноценная** — раньше была «поделка из того, что было»
