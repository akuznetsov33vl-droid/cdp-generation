# 5. Отраслевые карты коммуникаций (готовые механики)

> Это **готовые карты сценариев** из официальной таблицы CDP Calltouch «Доступные механики». Используйте их как основу при подготовке Дорожных карт и индивидуальных предложений. В каждой карте указаны: сценарий, триггер, контент, канал коммуникации.
>
> 💚 В оригинале **зелёным** выделены приоритетные сценарии. Здесь они помечены ⭐.

## Структура карты

Каждый сценарий описан четырьмя параметрами:
- **Сценарий** — что делаем
- **Триггер** — событие, запускающее коммуникацию
- **Контент** — текст сообщения / описание действия
- **Канал** — где отправляем

## 📊 Как использовать эти карты

### При составлении Дорожной карты для клиента

1. Определи отрасль клиента
2. Найди соответствующую карту
3. Выдели **3 приоритетных сценария** (помеченных ⭐) — это база для Дорожной карты
4. При необходимости расширь до 5-7 сценариев из той же карты
5. Если у клиента есть особенности — адаптируй или возьми сценарии из универсальной карты

### При проведении выявления потребностей

Используй сценарии из карты как **подсказки для вопросов**:
- «У вас бывает, что клиент возвращается на сайт, но не оставляет заявку?» → сценарий «Возвращается на страницы»
- «У вас есть программа повторных заказов?» → сценарий «Циклическая услуга»
- «Бывает, клиент бросает корзину/заказ?» → сценарий «Брошенная оплата»

### При продаже расширений

- Сценарий «**Выявлен интерес**» → продажа Скоринга
- Сценарий «WhatsApp» → продажа WABA
- Сценарии с офлайн-точками → продажа Big Data

## 🎯 Топовые сценарии каждой отрасли (для быстрого старта)

| Отрасль | Топ-3 сценария |
|---------|----------------|
| **Медицина** | Мотивация к приёму после записи + Возврат на страницы услуг + Циклическая услуга (чистка) |
| **Автодилеры** | Импорт офлайн-базы (визиты в салон) + Зависшая сделка после тест-драйва + Возврат после покупки в сервис |
| **Застройщики** | Возврат из воронки на сайт + Зависшая сделка после просмотра + Выявлен интерес |
| **E-com** | Брошенная корзина + Возврат без полезных действий + Подписка на рассылки |
| **Фитнес** | Мотивация после бесплатной тренировки + Возвращение без абонемента + Циклическое продление |
| **Образование** | Подписка на рассылки + Зависшая сделка после пробного + Мотивация к первому курсу |
| **Микрозаймы** | Мотивация к первому займу + Напоминание о погашении + Реактивация по займам |

| **Строительство домов** | Мотивация к лиду + Сделка после визита в офис + Зависшая сделка |
