# 1. Обзор продукта CDP Calltouch

## Что такое CDP Calltouch

**CDP Calltouch** (Customer Data Platform) — платформа клиентских данных, которая собирает всю информацию о клиентах из разных источников в единый профиль для дальнейшего персонализированного взаимодействия.

### Базовые определения

| Термин | Значение |
|--------|----------|
| **CDP** | Customer Data Platform, платформа клиентских данных |
| **Триггерные сценарии** | Функция внутри CDP для коммуникации с базой по событиям |
| **Старое название** | Смарт-коммуникации (Smart Communications) |
| **ПО в договорах** | Колтач Лидс (Calltouch Лидс) |
| **В счетах/актах** | «Доступ к платформе CDP (ПО Колтач Лидс)» |
| **Тип инструмента** | CRM-marketing |

## Главное позиционирование

> **«Платформа для роста лидов и продаж через персонализированные коммуникации»**

CDP — не «ещё один попап». Это **система**, которая:
- Объединяет данные о клиенте из всех источников (сайт, CRM, АТС, коллтрекинг, Big Data)
- Собирает 360° профиль клиента (демография + поведение + история обращений)
- Сегментирует аудиторию по 100+ параметрам
- Запускает персонализированные коммуникации в момент срабатывания триггера
- Использует разные каналы: попапы, email, SMS, мессенджеры, CRM-уведомления

## Бизнес-задачи, которые решает

### Пять болевых точек, на которые отвечает CDP

| № | Задача | Метрика влияния |
|---|--------|-----------------|
| 1 | Посетители слабо конвертируются в лиды | Рост CR в лид с сайта |
| 2 | Лиды не конвертируются в продажи | Рост CR из лида в продажу |
| 3 | Сделки долго движутся по воронке | Сокращение цикла сделки |
| 4 | Средний чек ниже расходов на привлечение | Рост среднего чека |
| 5 | Мало повторных продаж | Рост частоты повторных покупок |

### Три ключевых направления применения

#### 🎯 Направление 1: Рост конверсии сайта в уникальные обращения

- **Проблема клиента:** Конверсия менее 2%
- **Решение:** Персонализированные popUp на основе истории и поведения
- **Метрики:**
  - 1-3% — средний CR из PopUp в лид
  - +0,5% — прирост базы лидов
  - +0,1 п.п. — прирост CR визита в контакт

#### 🎯 Направление 2: Конвертация базы контактов в целевой лид

- **Проблема клиента:** Клиенты сравнивают конкурентов, теряются между визитами
- **Решение:** Реакция на возврат клиента или интерес к конкурентам через Big Data
- **Метрики:**
  - 5% лидов возвращаются на сайт ежемесячно
  - Клиенты в среднем сравнивают 3 компании при покупке

#### 🎯 Направление 3: Рост конверсии из лида в продажу

- **Проблема клиента:** Лиды теряются в воронке, не доходят до сделки
- **Решение:** Сопровождение лида до продажи через персональные сообщения
- **Метрика:** +20% рост CR при сопровождении

## Решение для разных ролей

### Для отдела продаж

| Сейчас (без CDP) | С CDP Calltouch |
|------------------|-----------------|
| Клиенты теряются между сайтом и CRM | Все данные в едином окне |
| Клиенты долго двигаются по воронке | Автоматический «дожим» |
| Нет повторных продаж и низкий LTV | Сегментация для возвращения |
| Сложно выявить «горячие» лиды | Определение готовности по поведению |
| План по продажам не выполняется | Реактивация и рост конверсии |
| Высокие расходы на обработку базы | Автоматизация без нагрузки |

### Для маркетолога

| Сейчас (без CDP) | С CDP Calltouch |
|------------------|-----------------|
| Посетители слабо конвертируются | Персонализированные попапы |
| Лиды не доходят до продаж | Только квалифицированные обращения |
| Нет полноценного портрета клиента | Единая клиентская база |
| Невозможно точно сегментировать | 100+ параметров для сегментации |
| Высокий ДРР / низкий ROMI | Уменьшение CPL и рост ROMI |
| Много ресурса на ручной запуск | Настройка под ключ + автоматизация |
