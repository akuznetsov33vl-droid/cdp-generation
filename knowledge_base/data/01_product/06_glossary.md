# 6. Глоссарий терминов

## Основные термины продукта

| Термин | Расшифровка |
|--------|-------------|
| **CDP** | Customer Data Platform — платформа клиентских данных |
| **Триггер** | Событие, запускающее сценарий (например, посещение сайта) |
| **Сценарий** | Цепочка автоматизированных коммуникаций |
| **Аудитория** | Сегмент клиентов для коммуникации |
| **Тегирование** | Присвоение метки лиду/сделке для аналитики |
| **Webhook** | HTTP-запрос для отправки данных во внешнюю систему |
| **siteID** | Уникальный идентификатор сайта в Calltouch |
| **Ретростата** | Историческая статистика по сценарию |
| **Окно ретроспективы** | Период, в который лид/сделка получают теги от сценария |

## Термины, связанные с продуктом Calltouch

| Термин | Расшифровка |
|--------|-------------|
| **Колтач Лидс** | Внутреннее название ПО CDP (для договоров) |
| **Big Data** | Инструмент для оцифровки пешеходного трафика и обогащения |
| **Скоринг** | Оценка заинтересованности аудитории |
| **WABA** | WhatsApp Business API |
| **Smart SMS** | Таргетированная SMS-рассылка |
| **ОЗ** | Обратный звонок (callback) |
| **УЗ** | Умная заявка |
| **Коллтрекинг** | Отслеживание звонков |
| **Сквозная аналитика** | От первого касания до ROI |

## Маркетинговые метрики

| Термин | Расшифровка |
|--------|-------------|
| **CR** | Conversion Rate — коэффициент конверсии |
| **CPL** | Cost Per Lead — стоимость лида |
| **CPA** | Cost Per Action — стоимость целевого действия |
| **LTV** | Lifetime Value — пожизненная ценность клиента |
| **ROI** | Return on Investment — окупаемость инвестиций |
| **ROMI** | Return on Marketing Investment — окупаемость маркетинговых инвестиций |
| **ДРР** | Доля рекламных расходов |
| **Лид** | Потенциальный клиент, проявивший интерес |
| **PopUp** | Всплывающее окно на сайте |
| **УТП** | Уникальное торговое предложение |
| **CRM** | Customer Relationship Management — система управления взаимоотношениями с клиентами |

## Технические термины

| Термин | Расшифровка |
|--------|-------------|
| **JS-цель** | JavaScript-событие на сайте, отслеживаемое в аналитике |
| **UTM-метки** | Параметры URL для отслеживания источников трафика |
| **Куки (cookies)** | Идентификатор пользователя в браузере |
| **Sha256** | Алгоритм шифрования для передачи зашифрованных контактов |
| **API** | Интерфейс для взаимодействия с системой |
| **PIXEL** | Скрипт отслеживания на сайте |

## Внутренние отделы и роли

| Сокращение | Расшифровка |
|------------|-------------|
| **ОБА / DBA** | Отдел Бизнес-Аналитики (Department Business Analytics) |
| **ОРНП** | Отдел регистрации новых проектов |
| **МПП** | Менеджер по подключению продуктов |
| **КС** | Клиентский сервис |
| **ОП** | Отдел продаж |
| **АС** | Аккаунтинг (Accounts) |
| **ДС** | Дополнительное соглашение |

## Сегменты аудитории

| Сегмент | Описание |
|---------|----------|
| **Без контактов** | Анонимный посетитель сайта |
| **С контактами** | Известный клиент (есть телефон/email) |
| **Новый лид** | Первое обращение клиента |
| **Повторный лид** | Уже обращавшийся клиент |
| **Горячий лид** | Готов к покупке (по поведению или скорингу) |
| **Холодный лид** | Изучает, ещё не готов |
| **Клиент** | Уже совершил покупку |
| **Возвращённый** | Клиент с историей, вернулся на сайт |

## Типы коммуникаций

| Тип | Описание |
|-----|----------|
| **Триггерная коммуникация** | Автоматически запускается по событию |
| **Разовая рассылка** | Единоразовая отправка по сегменту |
| **Каскадная коммуникация** | Серия касаний разными каналами |
| **Реактивационная** | Возвращение «спящих» клиентов |
| **Welcome-цепочка** | Серия писем после первого контакта |
| **Транзакционная** | Сообщение о статусе сделки |
| **Промо-коммуникация** | Маркетинговое предложение |
