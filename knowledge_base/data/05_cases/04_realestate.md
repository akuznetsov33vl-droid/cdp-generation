> Используются для прокачки доверия в предложениях для застройщиков, агентств недвижимости, риэлторских компаний.

## ГК «Точно» (девелопер)

### Задача
Оцифровать пешеходный трафик клиентов рядом с объектами и улучшить точность рекламных кампаний.

### Решение
- CDP с подключением **Big Data партнёров** Calltouch
- Оцифровка пешеходов рядом с офисами продаж и стройками

### Результат
- ✅ Оцифровка **82% пешеходного трафика**
- ✅ Повышение точности рекламных кампаний
- ✅ Снижение стоимости привлечения клиента

### Применимо для
- Застройщиков с офисами продаж
- Когда важна работа с пешеходным трафиком
- Премиум-сегмент недвижимости

### Цитата для предложения
> «У ГК "Точно" CDP в связке с Big Data оцифровал 82% пешеходного трафика — то, что в любых других системах остаётся "тёмной зоной".»

## Застройщик премиум-класса (обезличенный кейс)

### Задача
Из существующей базы 2 000 контактов выявить тех, кто прямо сейчас рассматривает покупку.

### Решение
- CDP + Скоринг (триггер «Выявлен интерес»)
- Точечные коммуникации с заинтересованными

### Результат

| Параметр | Значение |
|----------|----------|
| База контактов | **2 000** |
| Выявлено интересов за 1 месяц | **160** |
| Целевые лиды | **47** |
| Триггерные интересы (стоимость) | 15 000 ₽ |
| **Стоимость целевого лида** | **319 ₽** |
| Средняя стоимость целевого лида у застройщика | 25 000 ₽ |

### Эффект
- ✅ **CPL в 78 раз ниже** среднерыночного по премиум-недвижимости
- ✅ Из «спящей» базы выявлены **47 целевых лидов** за месяц

### Применимо для
- Застройщиков с накопленной базой
- Премиум-сегмента (квартиры от 10 млн ₽)
- Бизнеса, где целевой лид стоит десятки тысяч рублей

### Цитата для предложения
> «У застройщика премиум-класса целевой лид через CDP+Скоринг стоил 319 ₽ — в 78 раз ниже, чем средний CPL в премиум-недвижимости (25 000 ₽).»

## Релевантные сценарии для недвижимости

### Сценарий «Просмотрел планировки»
- Триггер: посетил 3+ страницы планировок одного типа
- Действие: попап с предложением консультации архитектора + персональный расчёт ипотеки

### Сценарий «Был в офисе, не подписал»
- Триггер: сделка статуса «Просмотр в офисе» → ожидание «Договор» 7 дней → не наступило
- Действие: webhook менеджеру + SMS с подарком (например, скидка на парковку)

### Сценарий «Вернулся через неделю»
- Триггер: повторное посещение сайта (был 7+ дней назад)
- Действие: webhook менеджеру в реальном времени + персональный попап

### Сценарий «Сравнивает застройщиков» (с Big Data)
- Триггер: клиент посещает сайты других ЖК
- Действие: приоритетный звонок + SMS с уникальным предложением

### Сценарий «Изменение цены на квартиру»
- Триггер: клиент следил за конкретным объектом → цена изменилась
- Действие: уведомление об изменении цены/появлении промо

### Сценарий «Реактивация по типу объекта»
- Аудитория: смотрели студии год назад
- Триггер: запуск нового корпуса со студиями
- Действие: персональное приглашение

## Что говорить, когда клиент-застройщик сомневается

### «У нас длинный цикл, CDP не сработает»
> Длинный цикл — это **идеально** для CDP. Клиент 3-12 месяцев сравнивает застройщиков. Каждое его возвращение на ваш сайт — момент истины. CDP ловит этот момент.

### «У нас CRM и так считает все сделки»
> CRM работает с теми, кто оставил заявку. CDP работает **до** заявки. Когда клиент изучает планировки и сравнивает — CRM не знает об этом. CDP знает.

### «У нас высокий CPL по контексту, бюджет большой»
> Это и хорошо. Кейс премиум-застройщика: CPL целевого лида через CDP — 319 ₽. У вас, скорее всего, в десятки раз больше. Один лид CDP = годовая абонка.

### «Big Data — это сложно и дорого»
> CDP даёт базу в 10 500 ₽, а Big Data подключается уже с накопленной аналитикой. Пешеходный трафик возле офиса продаж — это бесплатное расширение базы лидов.

## Особенности недвижимости (для презентации)

### Почему CDP здесь критичен

1. **Очень дорогая реклама** — Яндекс.Директ для премиум-недвижимости от 5-15 тыс. ₽/лид
2. **Большая база «дозревающих»** — клиент 6-12 месяцев выбирает
3. **Высокая конкуренция** — клиент сравнивает 5-10 ЖК
4. **Офлайн-офисы** — Big Data позволяет оцифровать визиты без QR
5. **Высокий чек** — одна сделка окупает CDP на 5-10 лет

### Целевые метрики

| Метрика | Целевое значение |
|---------|------------------|
| CPL целевого лида (премиум) | 300-1 000 ₽ через CDP |
| Конверсия из «выявленного интереса» в лид | 20-30% |
| Возврат сделки из спящей базы | 5-10% |
| Окупаемость с одной сделки | 10+ месяцев CDP |
