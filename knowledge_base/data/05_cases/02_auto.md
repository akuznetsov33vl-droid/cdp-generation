# 2. Кейсы в авто

> Используются для прокачки доверия в предложениях для автодилеров, автосалонов, автосервисов, дистрибьюторов.

## «Автополе»

### Задача
Конвертировать посетителей сайта в обращения, особенно тех, кто уже знаком с салоном (есть в базе).

### Решение
- SMS-рассылка по базе через CDP
- Реверсивный обратный звонок
- Триггеры на посетителей-возвращенцев

### Результат
- ✅ **+556 лидов**
- ✅ **CPL 254 ₽**

### Применимо для
- Дилеров с большой клиентской базой
- Многомарочных салонов
- Дилеров с активным сезонным трафиком

### Цитата для предложения
> «"Автополе" получил +556 дополнительных лидов через CDP при CPL 254 ₽ — это в 5-10 раз дешевле, чем средний CPL по контекстной рекламе в авто.»

## Релевантные сценарии для авто

### Сценарий «Тест-драйв не дошёл»
- Клиент записался на тест-драйв, не пришёл
- Триггер: новая сделка статуса «Запись на тест-драйв» → ожидание сделки «Тест-драйв проведён» 3 дня → не наступило
- Действие: SMS «Не получилось вчера? Перенесём на удобное время» + email с описанием авто

### Сценарий «Сравнение с конкурентом» (через CDP + Скоринг)
- Триггер: клиент посещал сайты других дилеров (через Big Data)
- Действие: приоритетный звонок менеджера + SMS с уникальным предложением

### Сценарий «Зашёл в калькулятор кредита»
- Триггер: просмотр страницы /credit/ или /trade-in/ + 60 секунд
- Если контакты есть → webhook менеджеру «изучает кредит, звоните»
- Если нет → попап «Рассчитаем индивидуальную программу за 5 минут»

### Сценарий «Сезонный апсейл»
- Аудитория: купили авто 1+ год назад
- Триггер: по расписанию (1 марта — для весеннего сервиса; 1 ноября — для зимней резины)
- Действие: персональное предложение по сервису

### Сценарий «Возвращение через 2-4 недели»
- Аудитория: были на сайте 14-30 дней назад, не купили
- Триггер: повторное посещение сайта
- Действие: webhook менеджеру + попап «Рассмотрим персональные условия»

## Что говорить, когда клиент-дилер сомневается

### «У нас уже большой бюджет в Авто.ру и Дроме»
> Это и хорошо. CDP работает с теми, кого вы уже привели — конвертируете 5%, а 95% уходят. Мы возвращаем эти 95%.

### «Мы делаем обзвоны через CRM»
> Обзвоны хорошо работают со сделками. CDP добавляет работу с теми, кто пока не оставил заявку, но уже готов — по поведению на сайте.

### «Сложно посчитать ROI на авто»
> На авто это считается прямо: средний чек 1,5 млн, CR 5%, абонка 10 500 ₽. Достаточно одной дополнительной продажи в полгода, чтобы окупить год CDP. По кейсам у наших клиентов получается 50-150 доп. лидов в месяц.

### «У нас и так длинные циклы»
> Длинный цикл — это идеальные условия для CDP. Клиент 1-3 месяца сравнивает дилеров, и важно поймать его в момент возврата на ваш сайт.
