# 1. Подходящие отрасли для CDP Calltouch

## Карта релевантности по отраслям

### 🟢 Зелёная зона (высокая релевантность)

| Отрасль | Почему подходит | Релевантные продукты |
|---------|-----------------|---------------------|
| **Авто (дилеры)** | Длинный цикл, высокий чек, цифровой трафик | CDP + Скоринг + Big Data |
| **Недвижимость** | Самый длинный цикл, очень высокий чек | CDP + Скоринг + Big Data |
| **Медицина (премиум)** | Имплантация, ортодонтия, операции | CDP + WABA |
| **Образование** | Сравнение программ, длинный выбор | CDP + Скоринг |
| **Финансовые услуги** | Кредиты, ипотека, инвестиции | CDP + Скоринг |
| **B2B-услуги** (консалтинг, юристы) | Длинные продажи, дорогие услуги | CDP |

### 🟡 Жёлтая зона (средняя релевантность, требует анализа)

| Отрасль | Условия применимости |
|---------|---------------------|
| **E-commerce** | Средний/высокий чек, есть корзина и личный кабинет |
| **Туризм** | Премиум-сегмент, не масс-маркет |
| **Фитнес** | Премиум-клубы, абонементы от 30к |
| **Бьюти-индустрия** | Косметология, не парикмахерские |
| **Ремонт и стройка** | Сложные/дорогие проекты |
| **Школы танцев / йоги / etc** | Зависит от ценника абонемента |

### 🔴 Красная зона (низкая релевантность)

| Отрасль | Почему не подходит |
|---------|-------------------|
| **Импульсные покупки** | Нет цикла принятия решения |
| **Дешёвые сервисы** (от 1-3к) | Не окупится |
| **Только B2B мегакрупный с длинными продажами** | Слишком мало цифровых событий |
| **Узкие ниши без сегментации** | Не на чем сегментировать |

## Детальный профиль для каждой отрасли

### 🚗 Авто

**Почему подходит:**
- Длинный цикл принятия решения (1-3 месяца)
- Высокий средний чек (от 1 млн ₽)
- Клиенты сравнивают 3-5 дилеров
- Есть классифайды (Авто.ру, Дром) для скоринга
- Есть тест-драйвы → конкретные триггеры

**Типовые задачи:**
- Возврат клиентов, не дошедших до тест-драйва
- Уведомления при возвращении на сайт после посещения конкурента
- Реактивация по базе после сезонного спада

**Примеры кейсов:** Автополе (+556 лидов), Avantime (+140 лидов), Арконт (ROMI 277%), Форсаж (+27% удачных звонков)

### 🏘️ Недвижимость

**Почему подходит:**
- Самый длинный цикл (3-12 месяцев)
- Самый высокий чек (от 5 млн ₽)
- Клиенты обходят 5-10 объектов
- Очень дорогая реклама (CPL от 5-15 тыс. ₽)
- Есть Big Data партнёры с данными о посещениях офлайн-офисов

**Типовые задачи:**
- Сигналы об уходе к конкуренту-застройщику
- Реактивация после просмотра в офисе
- Догрев между визитами объектов

**Примеры кейсов:** ГК Точно (оцифровка 82% пешеходов), застройщик премиум (47 целевых лидов из 160 «выявленных интересов» при CPL 319 ₽)

### 🏥 Медицина

**Почему подходит (для премиум-сегмента):**
- Дорогие профильные услуги (имплантация, операции, ортодонтия)
- Цикл принятия решения 2-4 недели
- Клиенты сравнивают клиники
- Есть страховой B2B-сегмент

**Ограничения для CDP:**
- Триггер «Выявлен интерес» подходит только для дорогих услуг
- НЕЛЬЗЯ рекламировать услуги, требующие спецподготовки (через рекламные SMS)
- Нельзя предполагать у клиента болезнь
- Реклама услуг врача может трактоваться как предположение о расстройстве

**Типовые задачи:**
- Догрев на странице имплантации/ортодонтии
- Реактивация после первичной консультации
- Серия писем для эстетических услуг (отбеливание, виниры)
- Болевые страницы → быстрая запись

**Примеры кейсов:**
- Сеть клиник «Вита» — +352 сделки на 444 623 ₽ без скидок
- «Вертеброцентр» — CPL с 1 389 ₽ до 308 ₽ (×4,5), повторные визиты ×2, ROI 198%

### 🎓 Образование

**Почему подходит:**
- Долгий выбор курса/программы
- Чек от 50к (для серьёзных программ)
- Сравнение нескольких школ
- Есть пробные уроки/тестовые недели

**Типовые задачи:**
- Догрев между пробным уроком и решением
- Серия писем с программой обучения
- Реактивация ушедших с пробного

### 💰 Финансовые услуги

**Почему подходит:**
- Сложные продукты (кредиты, ипотека, инвестиции)
- Долгое сравнение условий
- Высокая стоимость лида в рекламе
- Есть скоринг по интересу к кредитам

**Ограничения:**
- Нельзя многое в SMS (рекламные ограничения)
- Регулирование ЦБ
- Большие требования к KYC

### 🛒 E-commerce

**Когда подходит:**
- Средний чек от 5 000 ₽
- Есть личный кабинет и корзина
- Есть постоянная клиентская база (повторные продажи)
- Сезонные акции и распродажи

**Типовые задачи:**
- Брошенная корзина
- Cross-sell после покупки
- Реактивация спящих клиентов
- Wishlist с уведомлениями

**Примеры кейсов:**
- Garlyn — выручка +351 144 ₽, CPL -27% к Яндекс.Директ
- Кубань Инструмент — продажи +24, ROI 1 842%
- Магазин парфюмерии — 7 тыс. ₽ → 1,3 млн ₽ выручки
- Shemalini — +531 лид, ROI 19 559%

### ✈️ Туризм

**Когда подходит:**
- Премиум-сегмент (туры от 100к+)
- Долгое планирование (2-6 недель)
- Сравнение нескольких направлений
- Сезонность

**Типовые задачи:**
- Догрев между просмотром тура и бронированием
- Сезонные предложения
- Реактивация по сезонам

### 💪 Фитнес

**Когда подходит:**
- Премиум-клубы (абонемент от 30к/мес)
- Сложные форматы (персональные тренировки, реабилитация)
- Сезонность (январь, перед летом)

**Когда не подходит:**
- Бюджетные клубы со средним чеком 2-5к/мес — не окупится

**Типовые задачи:**
- Догрев после пробной тренировки
- Реактивация после паузы
- Big Data для пешего трафика рядом

## Как выбрать отрасль для предложения

### Шаг 1: Проверь чек

| Средний чек | Подходит CDP? |
|-------------|---------------|
| До 3 000 ₽ | ❌ Не окупится |
| 3 000 - 10 000 ₽ | ⚠️ Только если высокий LTV |
| 10 000 - 30 000 ₽ | ✅ Подходит |
| От 30 000 ₽ | ✅✅ Идеально |

### Шаг 2: Проверь цикл

| Цикл | Подходит CDP? |
|------|---------------|
| Импульсная покупка (минуты) | ❌ |
| 1-3 дня | ⚠️ |
| 1-2 недели | ✅ |
| 1+ месяц | ✅✅ |

### Шаг 3: Проверь трафик

| Трафик/мес | Подходит CDP? |
|------------|---------------|
| < 1 000 | ❌ Слишком мало для статистики |
| 1 000 - 10 000 | ⚠️ Окупится только при высоком чеке |
| 10 000 - 60 000 | ✅ Идеально |
| > 60 000 | ✅✅ Считается по 21 коп./посетитель |

### Шаг 4: Проверь конкурентность

| Конкуренция | Подходит CDP? |
|-------------|---------------|
| Монополия | ⚠️ Меньше пользы (клиенту некуда уходить) |
| 1-2 конкурента | ✅ |
| Высокая конкуренция | ✅✅ Big Data партнёров критична |

## Типовые цели клиентов по отраслям

| Отрасль | Главная цель | Ключевая метрика |
|---------|--------------|------------------|
| **Авто** | Возврат не купивших | CR в тест-драйв |
| **Недвижимость** | Удержание интересующихся | CR в просмотр объекта |
| **Медицина** | Конверсия на дорогие услуги | CR на консультацию |
| **Образование** | Конверсия после пробного | CR в покупку курса |
| **Финансы** | Снижение стоимости лида | CPL |
| **E-commerce** | Брошенная корзина | CR корзина → заказ |
| **Туризм** | Конверсия в бронирование | CR в бронь |
| **Фитнес** | Удержание после пробного | CR в абонемент |
